# PWA Icons

This directory contains the PWA icons for the Laravel Queue Management System.

## Required Icons

The following icon sizes are needed for the PWA manifest:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Shortcut Icons
- kiosk-96x96.png
- monitor-96x96.png
- login-96x96.png

## How to Generate Icons

1. Create a base icon (512x512 recommended) with a queue/number theme
2. Use an online PWA icon generator like:
   - https://walid50dev.github.io/icongen/
   - https://elwin.vdhazel.com/tools/pwa-icon-manifest-generator
   - https://mittl-medien.de/pwa-asset-generator

3. Upload your base icon and download the generated icon pack
4. Place the icons in this directory with the correct names
